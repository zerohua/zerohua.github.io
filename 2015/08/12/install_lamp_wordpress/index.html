Install LAMP + Wordpress

###Step One - Install Apache
sudo apt-get update (update ubuntu package list)
sudo apt-get install apache2

###Step Two - Install Mysql
# 不是必要的指令 

sudo apt-get install mysql-server php5-mysql
sudo mysql_install_db ( create it's database directory structure) #
sudo mysql_secure_installation (run a simple security script) #

Step Three — Install PHP

sudo apt-get install php5 libapache2-mod-php5 php5-mcrypt

Install WordPress on Ubuntu Apache

###Step One - Install unzip (for digitalOcean)
Linode 已自動安裝

sudo apt-get install unzip

###Step Two - Install WordPress SoftWare
cd /var/www
wget https://tw.wordpress.org/latest-zh_TW.zip

###Step Three - 修改 wordpress 資料夾名稱
更改為test.tw

mv wordpress test.tw <return>
cd test.tw <return>
mkdir logs <retrn> (新增logs資料夾)
cd .. <return>

Step Four - 設定wordpress 目錄權限

chown -R www-data:www-data test.tw <return>

Step five - 設定固定網址參數

sudo a2enmod rewrite <return>
cd test.tw <return>
touch .htaccess <return>
chmod 777 .htaccess <return>

**在Wordpress固定網址頁面更新您要的固定網址結構

再將「.htaccess」檔案存取權限變更為644

chmod 644 .htaccess <return>

Step Seven - 新增wordpress db

sudo mysql -uroot -pscum0923 <return>

mysql >

create database test; <return>
exit <return>

##Install Ubuntu Virtual Host

###Step One - 新增 Virtual Host
cd /
cd /etc/apache2/sites-available
nano test.tw.conf

test.tw.conf 內容 ctrl + o -->save ctrl + x -->exit

<VirtualHost *:80>
    ServerAdmin nono520@gmail.com
    DocumentRoot /var/www/test.tw/
    ServerName test.tw
    ServerAlias www.test.tw
    ErrorLog /var/www/test.tw/logs/error.log
    CustomLog /var/www/test.tw/logs/access.log combined
    <Directory /var/www/test.tw/>
            Options Indexes FollowSymLinks MultiViews
            AllowOverride All
            Order allow,deny
            allow from all
     </Directory>
</VirtualHost>

###Step Two - 使 test.tw 生效
sudo a2ensite test.tw.conf
sudo service apache2 restart
